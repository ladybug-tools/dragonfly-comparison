dragonfly_comparison.properties.room2d module¶

Room2D Comparison Properties.

class dragonfly_comparison.properties.room2d.Room2DComparisonProperties(host, comparison_floor_geometry=None, comparison_windows=None, comparison_skylight=None)[source]¶

Bases: object

Comparison Properties for Dragonfly Room2D.

Parameters:

  • host – A dragonfly_core Room2D object that hosts these properties.

  • comparison_floor_geometry – A single horizontal Face3D representing the floor plate of the Room2D to which the host Room2D is being compared. (Default: None).

  • comparison_windows – A list of WindowParameter objects that dictate the window geometries of the Room2D to which the host Room2D is being compared. (Default: None).

  • comparison_skylight – A SkylightParameters object that dictate the skylight geometries of the Room2D to which the host Room2D is being compared. (Default: None).

restore()[source]¶

Get a Room2D with host properties and geometry restored from the comparison.

The restored Room2D returned from this method will have all boundary conditions reset to outdoors and all shading parameters removed. Otherwise, all properties of the returned Room2D will match the host and all geometry will match the comparison.

property comparison_floor_geometry¶

Get or set a horizontal Face3D for the Room2D to which the host is compared.

If not set, all properties relating to floor geometry comparison will be zero (aka. unchanged).

property comparison_skylight¶

Get or set SkylightParameters describing how to generate skylights.

If not set, all properties relating to roof sub-face geometry comparison will be zero (aka. unchanged).

property comparison_windows¶

Get or set a tuple of WindowParameters describing how to generate windows.

If not set, all properties relating to wall sub-face geometry comparison will be zero (aka. unchanged).
